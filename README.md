# 🛒 Patalix Shopee App

**Patalix Shopee App** ek modern shopping application hai jo **Flipkart API** dwara powered hai. Iska maqsad users ko ek AI-driven aur surakshit shopping anubhav dena hai.

## ✨ Features (Tukdo mein)

* **Zara AI Assistant:** Ek anime-style smart assistant jo sahi deal chunne mein madad karti hai.
* **Flipkart Powered:** Sara data (Products, Price, EMI) Flipkart se live sync hota hai.
* **Smart Live Tracking:** Order ke nikalne se lekar delivery tak ki live map tracking.
* **Secure App Lock:** GPay ki tarah 4-digit PIN aur Fingerprint security.
* **No-Cost EMI:** Flipkart ke sabhi bank offers aur installment plans ka support.

## 🛠️ Technical Details

- **Language:** Python / JavaScript
- **API Sourcing:** Flipkart Affiliate API
- **UI Theme:** Premium White with Cobalt Blue accents.
- **Security:** In-built app-level encryption for user data.

## 🚀 Future Plans

- Voice-controlled shopping with Zara.
- Personalized price drop alerts.
